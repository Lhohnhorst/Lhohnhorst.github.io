    Textformen und thematische Vorschläge #

    Als Anregung für die letzte kreative Schreibaufgabe des Kurses wurde den Teilnehmenden eine Liste möglicher Textsorten und Themengebiete ausgehändigt, die im Folgenden dargestellt wird, um einen Überblick über die Rahmenbedingen des Projektes zu bieten. Die eingereichten Textprojekte finden Sie unter diesem Reiter (Texstprojekte) mit Überschriften, die von den Studierenden selbst gewählt wurden. Leider konnten etwaige Bilder aus urheberrechtlichen Gründen nicht aufgenommen werden, was insbesondere das erste Textprojekt betrifft.
